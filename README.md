
### `Intro`
Download all the songs on a give page on https://bsaber.com

#### `Install`
``` bash
npm install --save git+https://git@github.com/anzerr/bsaber.git
```

### `Example`
``` bash
git clone https://github.com/anzerr/bsaber.cli.git && \
cd bsaber.cli && \
npm i && \
npm link && \
bsaber --url "songs/top/?time=all"
```
